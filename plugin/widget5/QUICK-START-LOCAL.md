# 🚀 Quick Start: Test Locally in 3 Steps

## Step 1: Run Setup (2 minutes)

```bash
cd /home/kishank/ocean-plugin/plugin/widget5
./setup-local-test.sh
```

This will:
- ✅ Check prerequisites (Python, Node.js)
- ✅ Generate or download ocean data
- ✅ Install all dependencies
- ✅ Create startup scripts

## Step 2: Start Servers (30 seconds)

**Terminal 1 - xpublish Server:**
```bash
cd /home/kishank/ocean-plugin/plugin/widget5
./start-server.sh
```

**Terminal 2 - React App:**
```bash
cd /home/kishank/ocean-plugin/plugin/widget5
./start-app.sh
```

## Step 3: Test & Verify (1 minute)

**Terminal 3 - Run Tests:**
```bash
cd /home/kishank/ocean-plugin/plugin/widget5
./test-connection.sh
```

**Browser:**
- Open http://localhost:3000
- Open DevTools (F12)
- Check Console for: `✅ Metadata loaded`
- Watch particles animate! 🌊

---

## 🆘 Quick Troubleshooting

**Server won't start?**
```bash
pip install -r xpublish-server/requirements.txt
```

**React app won't start?**
```bash
npm install
```

**CORS errors?**
```bash
# Verify server is running:
curl http://localhost:9000/
```

**No data displayed?**
```bash
# Run connection test:
./test-connection.sh
```
